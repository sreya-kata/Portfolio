Covid-19 Vaccine Waste Tracker

Upon attending HackBeanpot I joined a team of 3 other Northeastern students. We dedicated the weekend towards creating an application that helps hospitals and other COVID-19 vaccine distribution centers to record and report vaccines that are wasted. This project was manifested as a result of the CDC reporting that many vaccines are going to waste, but there not being any concerted effort to track the waste.

The web application has three main components: the dashboard, login, and data entry page. The dashboard shows a table of top ten most "wasteful" counties (with the option to reverse the ordering). The graph on this page shows a daily tracker of the total number of vaccines wasted. The login page is a way od addressing security issues and administrative privileges. Lastly, if a successful login was entered, a data entry page comes into play with inputs for the hospital, county, date, and number of vaccines entered. This data, once submitted, is added to a database.

Using full-stack development principles, this project takes advantage of React components and API calls to a Java Sprint Boot backend server that is connected to a local SQL database.

Git

Git version control was taken full advantage of for this project as well all worked remotely and were not able to work as personally. Branches were committed to the GitHub to ensure minimum merge conflicts and pull requests were analyze thoroughly. I was able to utilize Git to its fullest potential here.

Spring Boot and SQL

The Java Spring Boot code called various SQL queries in many of its methods. Each method correlates to a different API endpoint for GET and POST requests. JSON values are added within these methods as well.

What's Next?

This project originally had 3 use cases in idea formation: hospitals, patients, and government officially. While our final product played more on the administrative side of things, we hope to add some patient portal aspect to allow patients to locate the nearest vaccine center correlated with waste.
