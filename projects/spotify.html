Spotify Playlist Curator

As a capstone project following my personal course in Python, I embarked on a project that would combined my interest in music with Python capabilties like Object-Oriented Programming and prior knowledge of APIs. The program connects a user's Spotify ID and Authorization Token (found here) as environment variables to retrieve data regarding recently played songs.

After displaying x amount of recent tracks, users have the option to make a playlist from them. Following this, a user can determine up to 5 tracks they would like to use as "seeds" for a recommended playlist of 50 new songs.

This project utilizes JSON unpacking with the official Spotify API and documentation to make GET and POST requests.
