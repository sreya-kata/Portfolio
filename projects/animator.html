Easy Animator

As a final paired-programming project in Object-Oriented Design, the task was to produce fully-functioning 2D-shape animations from simple .txt files. One of the main goals for this assignment was to utilize the MVC structure and stress the importance of where mutation is allowed.

Users are able to include commands regarding the input file (which must follow a specific format), a view type (text, SVG, or Swing), and optional speeds and output file. An additional view option (interactive) allows users to adjust the tempo, play state, loopback, and form of movement—continuous or discrete—while the animation is playing.

Using Object-Oriented Programming principles, this project takes advantage of abstraction, class delegation, and polymorphism. Swing elements like JFrames and JPanels were also used.

Interactive

In this setting, users can alter the number of ticks per second (by pressing "+" or "-"), pause and restart (by clicking buttons), enable loopback (through a checkbox), and turn on discrete mode (also through a checkbox).

Algorithm

An extra .txt file can be produced from a programmatic Bubble Sort Algorithm file included. Just as the algorithm itself, the program will continously switch each bar until they are in heigh ascending order.
