2024 Election Results

As my final project for data science, I teamed up with another classmate to produce a model that could predict county-level results for the upcoming presidential election, based on demographics.

We followed a pipeline that consisted of getting the data (largely from CSVs-Github for election data and census-related data for demographics), cleaning (extracting only relevant information from each CSV and then merging them into one dataframe), and doing the machine learning (we chose to use regression to estimate the proportion/votes for each party).

This project was a deeper delve into data science and machine learning concepts like regression analysis and predictions. We used the sklearn library to do our analytics and pandas to deal with dataframes.

Heatmap

We produced several plots in our analyses of the data. One significant one involved a seaborn heat/correlation map to plot relationships between various demographic features (for example: college graduation rate and % democratic vote).

Demographic Significance

Our initial idea invovled prediction election results purely from prior election results for a county. However, we wanted to incorporate demographic data so we revised our scope to focus on using demographic data to predict votes.
